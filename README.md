# Learn Macedonian 🇲🇰

A simple flashcard website to learn Macedonian vocabulary.

## How to use

1. Open `index.html` in your browser (double-click the file).
2. Tap or click a card to flip it and see the English translation.
3. Use **Previous** / **Next** or arrow keys to navigate.
4. Press **Got it ✓** to mark words you've learned (saved in your browser).
5. Filter by category or shuffle the deck with 🔀.

## Adding your own words

Edit `data.js` and add entries like:

```js
{ mk: "Мacedonian word", en: "English meaning", phonetic: "pronunciation hint", category: "greetings" },
```

Categories: `greetings`, `basics`, `food`, `numbers`, `phrases`

## Deploy online (optional)

Upload the folder to [GitHub Pages](https://pages.github.com/), [Netlify](https://www.netlify.com/), or [Vercel](https://vercel.com/) for free hosting.
